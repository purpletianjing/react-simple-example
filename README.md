# How to run this project

```
1. You must install npm and nodejs
2. run 'git clone git@github.com:purpletianjing/react-simple-example.git' in your terminal. You will see the project in your directory.
3. run 'cd react-simple-example' in your terminal
4. run 'npm install' in your terminal
5. run 'cd src/client' in your terminal
6. open index.html in your chrome 
7. you can see the cool effect
```
